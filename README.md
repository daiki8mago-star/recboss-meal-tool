# Recboss 食事プラン作成ツール

Recboss GYM コーチ用の減量サポート食事プラン作成ツール（1ファイル完結のHTML）。

クライアント情報を入力すると、Mifflin-St Jeor 式で代謝計算 → 摂取カロリー・PFC・3食/2食レイアウトを自動生成します。

## 主な機能

- **自動計算**: BMR / TDEE / 摂取目安 / PFC / 食事ごとの配分
- **安全診断**: 減量ペース（%/週）と摂取下限をチェックし、危険なら警告表示
- **メニュー編集＋自動栄養計算**: 「ごはん2杯」「鶏むね肉150g」のように食材＋分量で書くと、内蔵の食材データベースから kcal・PFC を自動計算（分量を変えると数値も連動）
- **間食欄（任意）**: 「＋ 間食を追加」で間食行を追加し、各食事の目安を軽減
- **クライアントを名前で保存**（このアップデートで追加）
  - 「名前」を入れて「この内容を保存」で端末（ブラウザ）に保存
  - 一覧から名前をタップで呼び出し、× で削除
  - 編集したメニュー内容も一緒に保存・復元
  - 入力内容は自動で一時保存され、次に開いたときに復元
  - 「バックアップ書出し / 読込」で別端末への移行や共有も可能（JSON）
- **PDF保存**: 「印刷 / PDF保存」ボタンからブラウザの印刷でPDF化

## 使い方

`index.html` をブラウザで開くだけです。サーバー不要・完全クライアントサイドで動作します。

> 保存データはブラウザの `localStorage` に保存されます。同じブラウザでのみ復元できます。
> 別の端末・ブラウザに移すときは「バックアップ書出し」でJSONを書き出し、移行先で「読込」してください。

## 栄養データの出典

メニュー欄の自動計算に使う食材データベースは
**「日本食品標準成分表2020年版（八訂）増補2023」** に準拠した値です（kcal は公式値、
P/F/C は八訂準拠）。`index.html` 内に約50品目を直接埋め込んでいるため、サーバー不要・
オフラインでも動作します。生鮮品は「生」基準で、調理により重量・kcal は多少変動します。

- `data/food_calories.json` … 出典データ（44品目・公式kcal・別名・状態つき）
- `tools/calorie_calc.py` … 上記JSONからカロリーを計算するPython参照実装

## 全食品（約2,478件）DBへの拡張方法

より網羅的にしたい場合は、公式Excelから全食品データベースを生成できます。

1. 文科省ページから「食品成分表(本表) Excel」をダウンロード
   <https://www.mext.go.jp/a_menu/syokuhinseibun/mext_00001.html>
2. `pip install openpyxl`
3. 列位置を確認: `python tools/build_full_db.py <Excel> --inspect`
4. 生成: `python tools/build_full_db.py <Excel> --name 3 --energy 5 --protein 6 --fat 8 --carb 11`
   → `food_calories_full.json`（kcal＋P/F/C）が出力されます

> `tools/build_full_db.py` は **エネルギーに加えて たんぱく質・脂質・炭水化物も抽出する** 版です。
